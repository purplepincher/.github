# PurplePincher `.github`

This repository exists so GitHub renders [`profile/README.md`](./profile/README.md) as the public landing page for the `purplepincher` organization.

`profile/README.md` is **not edited by hand here**. It is automatically synced from the main `README.md` on [`purplepincher/purplepincher`](https://github.com/purplepincher/purplepincher) by the [`sync-readme.yml`](./.github/workflows/sync-readme.yml) workflow, which runs every 30 minutes and can be triggered manually via `workflow_dispatch`.

If you want to change the org profile, edit the README in `purplepincher/purplepincher` instead.
